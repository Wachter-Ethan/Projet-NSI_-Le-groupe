# Projet-NSI_-Le-groupe
Projet destiner au concours trophée NSI.

Lors du lancement du jeu, on se retrouve face à un menu. En allant sur option, vous retrouverez les touches à utiliser pour jouer au jeu.

Les touches :
    Joueur 1 :
        Q : aller à gauche
        D : aller à droite
        Z : sauter
        E : attaque corp à corp
    Joueur 2 :
        K : aller à gauche
        M : aller à droite
        O : sauter
        P : attaque corp à corp

Attention : Les joueurs ne peuvent pas passer de l'autre coté du terrain. Celui qui apparait à gauche restera toujours à gauche de l'autre joueur et réciproquement pour celui qui apparait à droite.

En cas de lancement du projet en local, il est necéssaire de posséder la library python "pyxel". Toutefois, si vous ne la possédez pas, vous pouvez vous rendre sur pyxel studio afin de lancer le code sans soucis de library.
Faites attention à la presence de tout les fichier lors de l'exécution du code.
